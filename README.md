# Frontend Assessment Task

This project is a frontend UI implementation based on the provided Figma design.  
The focus of this task is clean UI, responsiveness, and proper component structure.

---

## Tech Stack

- Next.js
- Tailwind CSS
- Lucide Icons

---

## Features

- Pixel-perfect UI based on Figma design
- Fully responsive layout (mobile, tablet, desktop)
- Clean and reusable component structure
- Modern gradient, glow, and glassmorphism effects
- UI-focused implementation (no backend logic)

---
